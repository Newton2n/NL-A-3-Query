# Football Ticket Booking System

## Database Schema
* **Users Table:** Tracks user data .
* **Matches Table:** Tracks fixtures and pricing .
* **Bookings Table:** Links users to matches .

## Repository Contents
* `QUERY.sql`: Full schema, data insertions, and queris.
* `README.md`: Project documentation.

## Query 

| Query | Objective |
| :--- | :--- |
| **Query 1** | Get available Champions League matches |
| **Query 2** | Search users by name case-insensitively |
| **Query 3** | Handle empty payment statuses |
| **Query 4** | Join bookings with user and match details |
| **Query 5** | List all users including those without bookings |
| **Query 6** | Find bookings above average cost |
| **Query 7** | Get top 2 expensive matches skipping the highest |
